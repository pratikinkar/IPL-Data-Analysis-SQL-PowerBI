# IPL Data Analysis using SQL & Power BI

## Project Overview

This project analyzes IPL match and ball-by-ball data using SQL and Power BI.

## Tools Used

- SQL
- PostgreSQL/MySQL
- Power BI
- Excel

## Key Insights

- Team with most IPL trophies
- Highest win percentage
- Orange Cap winners
- Purple Cap winners
- Toss impact analysis
- Venue-wise performance
- Best strike rates
- Best economy rates
